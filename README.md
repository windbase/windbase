> **Note:** Windbase is currently under development and not yet ready for use.

Windbase is an open-source, web-based visual builder for Tailwind CSS built with Next.js. It enables developers and designers to create responsive user interfaces visually, with the ability to export clean, well-structured code.

## Features

- 🖱️ Drag-and-Drop Interface: Build layouts effortlessly by dragging components onto the canvas.
- 📚 Component Library: Access a variety of pre-built Tailwind CSS components.
- 🎨 Properties Panel: Customize component properties, styles, and content.
- 🔄 Real-Time Preview: Instantly see changes as you build your interface.
- 💾 Code Export: Generate and export clean React component code.
